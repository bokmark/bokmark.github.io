Java vm 面试题

jvm 内存结构 jmm的规则内存分为线程的工作内存、主内存之分 线程的工作内存包括虚拟栈、本地方法栈、程序计数器 主内存包括堆、方法区 堆按照垃圾回收算法分为 eden、from、to、老年代 方法区分为静态方法区，动态方法区 虚拟机栈是一个栈帧的filo栈。一个栈帧包含操作数栈、本地变量表、动态链接、返回地址 直接内存、本地内存 什么情况下内存栈溢出 线程爆炸多 描述一下 new一个对象的流程 检查对象的class是否存在方法区中，如果没有则加载 分配内存 分析内存存放的区域 逃逸分析：是否分配到栈上 是否大对象，如果是直接分配到老年代 分配到eden 内存的规整化 指针碰撞 空闲列表 考虑并发问题： talb：本地线程分配内存缓冲 cas：分配内存区域重试 内存空间初始化 设置 对象初始化 Java对象会不会分配在栈中
