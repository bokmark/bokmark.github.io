JMM

java memory model 现代计算机原理 cpu 多级缓存模型

  • 工作内存和主内存是抽象概念

  • 虚拟机中每个线程有一个工作内存，只有一个主内存

  • count++ 这个操作 在jmm中是怎么工作的

  • 每个线程只允许操作自己线程的工作内存不能操作主内存和别的线程的工作内存
